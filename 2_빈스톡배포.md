# 2. Beanstalk 배포하기

## 2-1. Github 연동

Github 연동을 ID & Password 방식으로 하면 보안에 민감하기 때문에 ssh 키로 연동 하겠습니다.  
jenkins가 설치된 서버에서 아래 명령어로 키를 생성합니다.

```bash
ssh-keygen -t rsa -f id_rsa
```

여기서 ```Enter passphrase```는 바로 엔터를 누르시면 됩니다.
